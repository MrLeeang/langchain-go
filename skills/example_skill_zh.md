# 数据分析技能（中文示例）

该技能用于在面对数据分析类任务时，为智能体提供一套清晰的分步流程指导。
当用户需要对一份数据集进行分析（例如 CSV、数据库表、日志文件等）时，可以使用本技能。  
智能体应根据本技能提供的步骤，有条理地完成数据收集、清洗、分析与结果汇报。

## 步骤

1. 明确分析目标和关键问题（例如：要回答什么问题、关注哪些指标）
2. 根据分析目标，从指定的数据源中收集所需数据（如调用相应的 MCP 工具获取数据）
3. 对原始数据进行清洗与预处理（处理缺失值、异常值，统一时间/单位等）
4. 根据目标选择合适的分析方法（描述性统计、对比分析、相关性分析等）
5. 在需要时生成可视化图表，以帮助理解数据模式与趋势
6. 总结关键发现，给出结论和可行的建议，并说明分析过程中的假设与局限性

## 使用建议

- 当用户说「帮我分析这份数据」「看看这组指标有什么问题」等类似需求时，优先考虑使用本技能。
- 在执行过程中，可以多次调用外部工具获取或处理数据，但请始终围绕上述步骤组织你的思考和输出。
